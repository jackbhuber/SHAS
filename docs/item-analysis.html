Validity Evidence for the Spiritual Harm and Abuse Scale (SHAS)

Item Analysis

This chapter documents our item analysis. This includes investigation of the following statistics from classical test theory: - Item difficulty - Item discrimination - Item-total correlations

We also examine item statistics from item response theory (IRT) including: - Item location parameters - Item characteristic curves (ICCs)

The goal of this information with which to improve items. One outcome of this analysis is insight into whether items have too many response categories.
